# Security Policy

This is a public education and research starter kit. It should not contain credentials, private data, account records, student records, client material, unpublished private drafts, or sensitive operational details.

## Reporting Issues

If you find a security issue in the public repo, open a GitHub security advisory or contact the maintainers through the repository's published security contact when available.

Do not include secrets, private data, or exploit instructions in a public issue.

## Data Boundary

Survival cards should use public-safe examples. Before sharing a card, remove names, private locations, account IDs, confidential documents, student details, proprietary workflows, and anything that could identify a vulnerable person or organization.

## Scope

The validator in this repo checks structure only. It does not certify that a card is accurate, safe, complete, legal, medically sound, financially appropriate, or operationally secure.
